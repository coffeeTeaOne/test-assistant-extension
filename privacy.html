🐞 七星瓢虫 隐私政策

最后更新：2026年6月6日

核心原则：七星瓢虫（Ladybug）是一款纯本地运行的 Chrome 扩展，不会收集、上传或分享您的任何个人信息。

1. 我们收集的信息

我们不收集任何个人信息。扩展运行过程中涉及的数据仅限于：

  • 录制的操作步骤：您在网页上的点击、输入等操作记录，仅保存在浏览器本地存储（chrome.storage.local）中
  • AI 配置：您自行填写的 API Key、Base URL 等，仅保存在浏览器本地
  • 登录配置档案：您主动保存的登录流程，仅保存在浏览器本地

2. 数据存储位置

所有数据均存储在您的浏览器本地（chrome.storage.local），不会上传到任何远程服务器。卸载扩展后，这些数据将被清除。

3. 第三方服务

扩展的 AI 对话功能需要调用您自行配置的大模型 API（如 OpenAI、Claude、DeepSeek 等）。这些 API 调用直接由您的浏览器发起，数据仅发送至您配置的 API 端点，我们不会介入或中转。

4. 权限说明

扩展申请的权限仅用于实现核心功能：

  • activeTab / scripting — 在当前标签页注入脚本以录制和回放操作
  • storage — 本地保存录制用例和配置
  • sidePanel — 提供侧边栏操作面板
  • downloads — 导出 JSON 文件到本地
  • <all_urls> — 支持录制任意网页的操作

5. 联系我们

如有隐私相关问题，请通过 GitHub Issues 反馈：https://github.com/coffeeTeaOne/test-assistant-extension/issues

🐞 Ladybug Privacy Policy

Last updated: June 6, 2026

Core Principle: Ladybug is a purely local Chrome extension that does not collect, upload, or share any of your personal information.

1. Information We Collect

We do not collect any personal information. Data involved during extension operation is limited to:

  • Recorded action steps: Your click, input and other web operations, stored only in browser local storage (chrome.storage.local)
  • AI Configuration: API Key, Base URL etc. that you fill in yourself, stored only locally in browser
  • Login Profiles: Login flows that you actively save, stored only locally in browser

2. Data Storage

All data is stored in your browser locally (chrome.storage.local) and will not be uploaded to any remote server. Data will be cleared when the extension is uninstalled.

3. Third-Party Services

The AI chat feature requires calling the LLM API that you configure yourself (such as OpenAI, Claude, DeepSeek, etc.). These API calls are initiated directly by your browser, and data is only sent to the API endpoint you configured. We do not intercept or relay.

4. Permissions

Permissions requested are solely for core functionality:

  • activeTab / scripting — Inject scripts into current tab for recording and playback
  • storage — Save recordings and settings locally
  • sidePanel — Provide sidebar panel
  • downloads — Export JSON files to local disk
  • <all_urls> — Support recording operations on any webpage

5. Contact Us

For privacy-related questions, please open a GitHub Issue: https://github.com/coffeeTeaOne/test-assistant-extension/issues
